# 🎬 StreamSavvy

A modern **React-based Movie Streaming Web Application** built with **Vite**, **React Router**, **Tailwind CSS**, and **JSON Server**. StreamSavvy provides an interactive movie browsing experience with authentication, favorites, search, dark mode, and responsive design.

---

## 🚀 Features

- 🎥 Browse latest movies
- 🔍 Search movies instantly
- ❤️ Add and remove favorites
- 👤 Mock Login Authentication
- 🌙 Dark & Light Theme
- 📱 Fully Responsive UI
- 🎬 Movie Details Page
- ▶️ Video Player Interface
- 🔄 REST API using JSON Server
- ⚡ Fast development using Vite

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/StreamSavvy.git
cd StreamSavvy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start JSON Server

```bash
npm run server
```

This starts the backend at:

```
http://localhost:5000
```

### 4. Start the development server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```
