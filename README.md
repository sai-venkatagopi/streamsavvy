## вҡҷпёҸ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/StreamSavvy.git
cd StreamSavvy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start JSON Server

```bash
npm run server
```

This starts the backend at:

```
http://localhost:5000
```

### 4. Start the development server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```
